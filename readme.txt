настройка ключевых клов -> config/theme.csv
настройка параметров скрипта -> config/config.csv:
	FILE_NAME - путь к файлу, который требуется обработать (достаточно названия файла, если он находится в папке classifier),
		поддерживаемые форматы - csv, xls, xlsx
	TEXT_FIELD_NAME - название колонки в обрабатываемом файле, которая содержит текст новости (например Text_OFD или u_summary)
запуск скрипта -> classify.bat (после запуска может потребоваться нажать Enter)
